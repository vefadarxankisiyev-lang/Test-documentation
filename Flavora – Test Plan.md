# Flavora – Test Plan

---

## 1. Project Name
**Flavora Website**

---

## 2. Project Description
Flavora interaktiv web platformadır və istifadəçilərə aşağıdakı imkanları təqdim edir:

- Resept paylaşmaq  
- Cooking session-lar yaratmaq  
- Digər istifadəçilərlə birlikdə yemək bişirmək  

Platforma yalnız sosial cooking təcrübəsi ilə məhdudlaşmır, həm də istifadəçilərə:

- Sağlam qidalanma  
- Balanslı pəhriz  
- Sağlam həyat tərzi  

ilə bağlı faydalı məlumatlar və reseptlər təqdim edir.

---

## 3. Test Objectives
Saytın funksional və qeyri-funksional xüsusiyyətlərinin tələblərə uyğun işlədiyini və istifadəçi təcrübəsinin optimal olduğunu yoxlamaq.

---

## 4. Test Scope

### Test ediləcək modullar:
- Home Page  
- Profile Page  
- Recipes Page  
- Favorite Page  
- Session Page  

---

## 5. Out of Scope
